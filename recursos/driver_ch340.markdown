Las placas compatibles con Arduino (en nuestro caso algunas NODEMCU) reemplazan el tracional microcontrolador FT232RL o el ATMEGA16u2 por un circuito integrado de origen Chino conocido como CH340. El driver CH340 es el software requerido para operar el circuito integrado de la interfaz USB en varios sistemas operativos (Windows, Linux, MAC, etc.).
